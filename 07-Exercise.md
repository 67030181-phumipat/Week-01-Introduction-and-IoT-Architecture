# **Workflow ที่นักศึกษาต้องทำ**

1. Fork repo รายวิชา
2. ทำความเข้าใจเนื้อหาที่เตรียมไว้
3. เลือกโจทย์ 3 จาก 6 ข้อ (มี 3 หมวด หมวดละ 2 ข้อ)
4. เปิด AI → ใช้ prompt มาตรฐานที่ให้ไว้ในแต่ละหวด
5. ตอบคำถามด้วยภาษาของตัวเอง
6. ให้ AI TA ตรวจและให้คะแนน
7. คัดลอกบทสนทนาใส่ `conversation.md`
8. Commit + Push
9. ส่ง Pull Request มายัง repo รายวิชา
